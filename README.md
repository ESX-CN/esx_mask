[安装教程]

1) cd 至你的 /[esx] 文件夹
2) 使用下列命令克隆
```
git clone https://github.com/ESX-CN/esx_mask esx_mask
```
3) 导入 `esx_mask.sql` 至你的数据库
4) 添加如下至你的 `server.cfg` 文件:
```
start esx_mask
```
